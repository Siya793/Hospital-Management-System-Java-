# Hospital-Management-System-Java-

I Develop a Hospital management System Using Java  Programming Langauge And Mysql for Database Connectivity.

int his project i created 3 Java  classes which are Doctor class,Patient class and HospitalManagementDriver Class.
here Doctor class will handle all the operations related to Doctor e.g - Doctors Availability,booking appointment etc.
Patient class is for handle all patient related operations like register patient name,take appointment for which doctor they want etc.
and lastly Driver class is use to handle all database related operation.
